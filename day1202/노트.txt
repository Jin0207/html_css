박스모델
	블록레벨 요소
		요소를 삽입했을 때, 혼자 한 줄을 차지하는 요소 
		요소 너비가 100%
		width, height가 존재
		EX) <div>, <p>
		
	인라인 레벨 요소
		한 줄을 차지하지 않는 요소
		화면에 표시되는 콘텐츠만큼만 영역을 차지, 나머지 공간에는 다른 요소가 올 수 있음
		width,height 없음 ==> 자체적으로 가지고있는 인라인레벨요소태그도 존재함 --img
		EX> <img>, <strong>, <span>
		
	*width, height
		수치(px)		고정 크기
		백분율 			브라우저창에따라 크기가 가변적
	
	display
		inline			인라인레벨요소로 배치
		block			블록레벨요소로 배치
		inline-block	다른요소를 옆에다가 배치할 수 있으면서도 가로,세로를 가질 수 있다.
	
	margin: 위 오른쪽 아래 왼쪽
			위 오*왼  아래
			위*아래 오*왼
			모든방향
			
		요소의 바깥 여백 설정
	
	
	padding: 위 오른쪽 아래 왼쪽
			 위 오*왼  아래
			 위*아래 오*왼
			 모든방향
			 
		요소의 안쪽 여백 설정

CSS 포지셔닝
	
	box-sizing
		content-box
			width, height 속성 값을 콘텐츠 영역 너비 값으로 사용 --기본값
	
		border-box
			 width, height 속성 값을 콘텐츠 영역에 테두리까지 포함한 박스 모델 전체 너비 값으로 사용
	
	float
		left
			해당 요소를 문서의 왼쪽으로 배치
		right
			해당 요소를 문서의 오른쪽으로 배치
		none
			좌우 어느 쪽으로도 배치하지 않음
	
	clear
		float속성을 무효화 시키는 속성
		한번 float가 설정되면 그 이후의 요소도 float의 영향을 받음
		
		none/ left/ right/ both
	
	---
	
	position
		staitc			
			요소를 문서의 흐름에 맞추어 배치 --기본값
			=> left, top 위치 지정 불가
		
		relative		
			바로 직전 요소에 따라 자연스럽게 연결해 배치하되 위치 지정 가능
			상대적인 위치를 사용하기에 다른 요소와 조화를 이룬다.
			top, left로 위치지정가능
		
		absolute		
			절대적인 위치
			문서의 흐름과는 상관없이 원하는 위치에 요소배치
			요소의 위치는 가장 가까운 부모요소나 조상 요소 중 position: relative인 요소
			없다면, 화면의 좌측상단(0,0)을 기준으로 원하는 위치 지정해 배치
			left,top,right, bottom 속성을 사용해 네 모서리에서 얼마나 떨어져 있는지 지정
			
		fixed			지정한 위치에 고정해 배치, 화면에 요소가 잘릴 수도 있다.
